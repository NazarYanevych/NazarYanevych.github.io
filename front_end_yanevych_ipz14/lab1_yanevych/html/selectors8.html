Приклади CSS Селекторів

1. Селектор (Universal Selector `*`): цей абзац для загального стилю всіх елементів на сторінці.

2. Селектор (Universal Selector для дітей `#container *`): всі дочірні елементи контейнера з обрамленням.

  • 3. Селектор (Descendant Selector `li a`): посилання в списку без підкреслення.
  • Посилання
  • 4. Селектор (Adjacent Sibling Selector `ul + p`): абзац після списку стане червоним.
  • 5. Селектор (Child Selector `div > ul`): список в контейнері має обрамлення.
    • Вкладений список, який не вибирається даним селектором.

6. Селектор (General Sibling Selector `ul ~ p`): абзаци після списку будуть червоними.

7. Селектор (Attribute Selector `[title]`): посилання з атрибутом `title` будуть зеленими. 8. Селектор (Attribute Selector з конкретним значенням `[href="http://everstudent.ua"]`): посилання на Everstudent стане золотим.
