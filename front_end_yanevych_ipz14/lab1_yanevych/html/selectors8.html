Основні CSS-селектори

1. Селектор (ID `#unique`): цей абзац має унікальний ID.

2. Селектор (Клас `.highlight`): цей абзац має клас highlight.

  • 3. Селектор (Теговий `li`): Елемент списку 1
  • 3. Селектор (Теговий `li`): Елемент списку 2
  • 3. Селектор (Клас `.highlight`): Елемент списку 3

4. Селектор (Дочірній `div > p`): Абзац всередині `div`.

4. Селектор (Дочірній `div > p`): Ще один абзац всередині `div`.

Заголовок

5. Селектор (Сусідній `h2 + p`): цей абзац йде після `h2`.

6. Селектор (`:hover`): наведи, щоб змінити колір

8. Селектор (`::first-letter`): Цей текст демонструє зміну першої літери.
